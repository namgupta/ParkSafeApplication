Setting up the project:

1. ParkSafeAndroidApplication - (Android Studio Project) import this as a project to Android Studio. The only modification needed is to LocateParking.java, set value of String server to your local server.

2. PHP files should be uploaded to your htdocs. Create a folder ParkSafe under htdocs and place all the php files there.

3. Import the SQL file to your MYSQL.

4. Import files of the WebAdminConsole to your htdocs.
